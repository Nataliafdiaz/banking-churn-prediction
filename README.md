# Predicción de Churn Bancario con Machine Learning

Proyecto de Machine Learning orientado a la predicción de abandono de clientes en el sector bancario.

## Objetivo

Desarrollar modelos predictivos capaces de identificar clientes con riesgo de abandono, permitiendo anticipar acciones de retención y apoyar decisiones comerciales basadas en datos.

## Contexto de negocio

En el sector financiero, la pérdida de clientes impacta directamente en la rentabilidad y en la relación a largo plazo con la entidad.  
Este proyecto busca analizar patrones de comportamiento y construir modelos que permitan detectar señales tempranas de churn.

## Tecnologías utilizadas

- Python
- Pandas
- NumPy
- Scikit-learn
- Machine Learning
- Jupyter Notebook
- GitHub

## Contenido del proyecto

- Análisis exploratorio de datos
- Modelo baseline con Regresión Logística
- Optimización con Gradient Boosting
- Aprendizaje no supervisado para segmentación
- Análisis complementario de crédito
- Interpretación de resultados orientada a negocio

## Archivos principales

- `1_EDA_Regresion_Logistica.ipynb`: análisis exploratorio y modelo inicial
- `2_GradientBoosting_Optimizacion.ipynb`: optimización del modelo predictivo
- `3_AprendizajeNoSupervisado.ipynb`: segmentación y análisis no supervisado
- `4_Extra_credit.ipynb`: análisis complementario
- `Churn_Modelling (1).csv`: dataset de trabajo
- `dataset_final_avance1.csv`: dataset procesado

## Enfoque de negocio

El proyecto combina análisis predictivo con conocimiento del sector financiero para transformar datos en insights accionables orientados a la retención de clientes.
